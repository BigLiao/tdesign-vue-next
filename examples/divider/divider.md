### 属性配置
| 属性        | 类型    | 默认值     | 必传 | 说明                                                         |
| ----------- | ------- | ---------- | ---- | ------------------------------------------------------------ |
| type        | string  | horizontal | N    | 水平还是垂直类型                                             |
| dashed      | boolean | false      | N    | 是否虚线，仅在水平分割线有效                                 |
| orientation | string  | center     | N    | 文本位置，左、中、右，可选值为 left / right / center（默认），仅在水平分割线有效 |
