### 自定义样式

`overlayStyle` 既可以作为 Object 使用，也可以作为 Function 动态使用。
希望浮层元素和触发元素宽度保持一致时，可以这样使用 `:overlayStyle="triggerElem => ({ width: triggerElem.offsetWidth + 'px' })"`
